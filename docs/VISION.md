# Product Vision

## Problem
WordPress and WooCommerce dashboards feel complex and intimidating for non-technical clients.

## Solution
A SaaS dashboard that hides WordPress complexity and lets clients manage products, orders, and blog content easily.

## Target Users
- Non-technical store owners (WooCommerce)
- Blog owners
- Agency clients who want a simpler experience

## Core Promise
Manage a WordPress site without using wp-admin.

## Product Principles
- Simplicity first
- Safe by default
- Fast everyday actions
- Clear separation between Owner and Client experiences

## Language & UI Requirements
- The product must be bilingual: English and French.
- UI language switcher available to users.
- Theme modes: System, Dark, and Light.
